We want to test the proposed end to end journey before moving into beta. It’s based on the application journey we tested previously.

We’re going to allow the users to have time before the usability session to read through the guidance, which has been updated based on previous research. We want to ensure that people understand the application journey is started online but involves submitting documents after.
